Welcome to my Top Ten HTML Tags / CSS Styles

HTML

1. Audio Tag

The audio Tags allows you to embed an audio clip into a web page. You can also provide the tag with some attributes like controls, autoplay, muted and loop. Controls will give you the ability to manipulate the audio. Autoplay well, speaks for itself. Muted will have the music muted by default, while loop will replay the audio once it finishes

2. Video Tag

The video Tag will allow you to embed a video clip into a webpage, and much like audio also allows for attributes such as controls, autoplay, and loop

3. Input Tag

The Input tag defines an interactive control within a web form. Some examples would be input type= "text", "email","number","checkbox","radio","submit","date" and so much more.








4. Image Tag

The Img tag defines an image inserted in the webpage

5. Link Tag

The link tag defines a link between the current web page and an external link or resource. href

CSS

1. Border-Color

The first CSS style, I'd like to mention is border color. Accompanied by border: Solid so that its visible

2. Padding

Next, is padding. There are several options to choose from, but if you do padding, it'll affect every side equally although you can do padding-left,-right,-top,-bottom but also a couple of more

3. Margin

Next is margin. Like padding, where you can default all sides, or can be specific with anyside

4. Text-Align

A person favorite of mine, is text-align. Where you can select text-align: left, right, center and justify

5. Display

Display, allows you to set the display behavior of the element. Some examples include, grid, flexbox, block, and none. these will allow you give your webpage responsiveness
